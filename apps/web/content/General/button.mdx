# Button

To trigger an operation.

## When To Use

A button means an operation (or a series of operations). Clicking a button will trigger its corresponding business logic.

In Hotdog-Ui we provide 6 types of button.

- üîµ Default button: used for the main action, there can be at most one default button in a section.
- ‚ö™Ô∏è Ghost button: used for a series of actions without priority.
- üî≥ Outline button: used for actions with a clear boundary.
- üò∂ Dashed button: commonly used for adding more actions.
- üî§ Text button: used for the most secondary action.
- üîó Link button: used for external links.

And we provide 2 animations for buttons.

- üèÉ‚Äç‚ôÇÔ∏è Ripple: creates a wave effect on click.
- üèÉ‚Äç‚ôÄÔ∏è Pulse: makes the button grow and shrink continuously.
